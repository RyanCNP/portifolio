# Portifolio

Este repositório contém o portfólio pessoal desenvolvido em Next.js com TypeScript, utilizando componentes personalizados e estilos globais.

## Sobre o projeto

O site apresenta seções como sobre mim, contato, educação, projetos e outros elementos interativos, todos construídos com componentes reutilizáveis.

## Tecnologias utilizadas

- Next.js
- React
- TypeScript
- Tailwind CSS (presumido pelos arquivos de configuração)

## Aviso

Este README foi gerado com o auxílio do GitHub Copilot (modelo Raptor mini Preview) e também contou com suporte da plataforma Vercel para hospedagem e implantação contínua.

## Uso

1. Instale as dependências:

   ```bash
   npm install
   ```

2. Execute o servidor de desenvolvimento:

   ```bash
   npm run dev
   ```

3. Acesse `http://localhost:3000` no navegador.
